Initial public release. Headless Phylax RPC switch library: EIP-6963 wallet detection, credible-require preflight, and assisted EIP add/switch/verify, with `react`, `wagmi`, `viem`, and `ethers` adapters plus a manual-add modal. Low-level utilities are exposed under the `/advanced` subpath so the root API stays narrow. Public types use branded domain primitives and discriminated-union results. Published as ESM-only, Node 22+.
